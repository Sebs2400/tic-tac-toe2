# Tic-Tac-Toe Project v1.0

A simple iteration of the Tic-Tac-Toe game, as my first portfolio project.

As this is my first project, I have tried to implement some basic coding in the creation of this terminal game.
I used mostly a while loop to check that all runs will the game is not over, with a combination of for loops, lists and if statements.
Need to polish the structure of the code some more, perhaps add some classes and methods.
Added an is_tie() method to check for draws if neither player wins.
Added a change_player() method so the program knows that it has to change players after every valid move. 

## Setup
1. Clone the repo.
2. Ensure you have the latest Python version installed.
3. Run 'Python main.py' to run the program.
